# Fashion Influencer Matcher - Frontend

## 개발 환경 설정

### 1. 의존성 설치

```bash
cd frontend
npm install
```

### 2. 환경 변수 설정

```bash
cp .env.example .env.local
# .env.local 파일 편집
```

### 3. 개발 서버 실행

```bash
npm run dev
```

- 개발 서버: http://localhost:3000
- API 프록시: http://localhost:8000

## 빌드

```bash
npm run build
```

## 기술 스택

- React 18
- TypeScript
- Vite
- TanStack Query
- TanStack Table
- Tailwind CSS
- React Router
- Lucide React (icons)
