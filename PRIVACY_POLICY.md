# 개인정보처리방침 (Privacy Policy)

**시행일:** 2026년 2월 19일

## 1. 개요

Fashion Influencer Matcher(이하 "서비스")는 소규모 패션 브랜드와 인플루언서 마케팅 에이전시가 데이터 기반으로 인플루언서를 선정할 수 있도록 돕는 SaaS 플랫폼입니다. 본 개인정보처리방침은 서비스 이용 과정에서 수집되는 데이터의 처리 방침을 명시합니다.

## 2. 수집하는 데이터

### 2.1 계정 정보
- 이메일 주소
- 비밀번호 (bcrypt로 암호화하여 저장)
- 계정 생성일

### 2.2 Instagram 데이터 (Business Discovery API)
서비스는 Instagram Graph API를 통해 **공개 비즈니스/크리에이터 계정**의 데이터만 수집합니다:

**프로필 데이터:**
- Instagram username
- 프로필 이름
- 팔로워 수
- 팔로잉 수
- 게시물 수
- 프로필 바이오
- 웹사이트 URL
- 인증 여부 (verified status)

**미디어 데이터 (최근 20개 게시물):**
- 게시물 캡션
- 해시태그
- 댓글 수
- 좋아요 수 (제공되는 경우)
- 미디어 타입 (IMAGE/VIDEO/CAROUSEL)
- 게시물 URL
- 게시 시간

### 2.3 분석 데이터
- 브랜드와 인플루언서 간의 유사도 점수
- 참여율 (engagement rate)
- 카테고리 분류 결과
- 협업 탐지 결과

### 2.4 수집하지 않는 데이터
서비스는 다음 데이터를 **수집하지 않습니다**:
- 비공개 계정 정보
- 개인 계정의 팔로워 목록
- 사용자의 개인 메시지 (DM)
- 계정 비밀번호
- 위치 정보
- 연락처 정보

## 3. 데이터 수집 목적

수집된 데이터는 다음 목적으로만 사용됩니다:

1. **인플루언서 분석**: 브랜드와 인플루언서 간의 적합도 분석
2. **참여율 계산**: 인플루언서의 평균 참여율 산정
3. **카테고리 분류**: 패션 카테고리 자동 분류
4. **결과 제공**: 분석 결과 대시보드 제공

## 4. 데이터 보유 기간

| 데이터 유형 | 보유 기간 | 삭제 시점 |
|------------|----------|----------|
| 사용자 계정 정보 | 계정 삭제 시까지 | 계정 탈퇴 즉시 |
| Instagram 프로필 데이터 | 90일 | 수집 후 90일 경과 |
| 미디어 데이터 | 90일 | 수집 후 90일 경과 |
| 분석 결과 | 90일 | 분석 완료 후 90일 경과 |
| 캐시 데이터 | 6시간 (프로필), 1시간 (미디어) | TTL 만료 시 |

**자동 삭제**: 보유 기간이 만료된 데이터는 자동으로 삭제됩니다.

## 5. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

### 5.1 데이터 접근권
- 자신의 계정 정보 조회
- 분석 작업 내역 조회

### 5.2 데이터 삭제권
사용자는 언제든지 다음을 요청할 수 있습니다:
- 계정 및 모든 관련 데이터 삭제
- 특정 Instagram 프로필 데이터 삭제
- 분석 결과 삭제

**삭제 요청 방법:**
- 이메일: privacy@fasion.app
- 처리 기간: 7영업일 이내

### 5.3 데이터 처리 제한권
- 프로필 분석 결과 캐싱 비활성화 요청
- 특정 계정 분석 제외 요청

## 6. 데이터 공유 및 제3자 제공

서비스는 수집된 데이터를 **제3자에게 제공하지 않습니다**.

단, 다음 경우는 예외로 합니다:
- 법적 의무가 있는 경우 (법원 명령 등)
- 서비스 제공에 필요한 데이터 처리 업체 (AWS, Redis Labs 등)
- Instagram/Meta 플랫폼 (API 사용을 통한 데이터 수집)

## 7. 데이터 보안

### 7.1 기술적 보호 조치
- 모든 데이터 전송은 TLS 1.2+ 암호화
- 데이터베이스 암호화 (AES-256)
- API 키는 안전한 저장소에 보관
- 정기적인 보안 감사

### 7.2 관리적 보호 조치
- 최소 권한 원칙 적용
- 접근 로그 기록
- 정기적인 직원 보안 교육

## 8. Meta Developer Policy 준수

서비스는 Meta Developer Policy를 준수합니다:

- **Business Discovery API**를 통해서만 공개 데이터 수집
- **비즈니스/크리에이터 계정**만 분석 대상
- **프라이버시 정책** 명시 및 사용자 동의 획득
- **데이터 삭제** 기능 제공
- **보안** 조치 시행

## 9. 국제 데이터 전송

서비스는 AWS (Amazon Web Services)를 사용하며, 데이터는 **아시아 태평양 (서울) 리전**에 저장됩니다.

## 10. 쿠키 및 추적 기술

서비스는 다음 목적으로 쿠키를 사용합니다:
- 세션 관리 (JWT 토큰 저장)
- 사용자 선호 설정 저장

**쿠키 거부**: 브라우저 설정에서 쿠키를 비활성화할 수 있으나, 서비스 이용에 제한이 있을 수 있습니다.
