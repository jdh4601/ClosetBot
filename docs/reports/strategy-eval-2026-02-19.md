## 세부 평가

### 1. BM 명확성 — 7/10 (가중치 20%)
타깃(소규모 패션 브랜드)·문제(시간/정확도)·가치제안(5분 내 데이터 기반 비교)은 명확함. 다만 지불 주체/요금제/결제 방식이 명시되지 않아 과금 플로우는 추정 수준.

### 2. 가격 전략 & WTP — 4/10 (가중치 25%)
가격대, 요금제 구조, WTP 검증 근거 없음. 경쟁사 벤치마크나 인터뷰 데이터 부재. 한국 SMB 지불 한계(월 ₩15만–₩60만) 내 포지셔닝 계획 필요.

### 3. CAC/LTV 추정 — 3/10 (가중치 25%)
획득 채널, 전환 퍼널, 이탈률 가정이 없어 LTV:CAC 계산 불가. 초기 베타/레퍼런스 고객 확보 코스트와 유지 기간 가정이 필요.

### 4. 성장 경로 (0→1→N) — 5/10 (가중치 20%)
기술 로드맵은 탄탄하나, 0→1 채널(커뮤니티/파트너/아웃바운드)과 10→100 스케일 메커니즘(PLG, 리퍼럴, 콘텐츠)이 비어 있음. 경쟁대비 차별화(공식 Graph API 준수, 투명 근거) 포지셔닝은 가능하나 GTM 설계 필요.

### 5. 수익 마일스톤 — 3/10 (가중치 10%)
MRR 타깃, 유료 전환율, 손익분기점 시점 등의 수치/일정이 없음. 인프라 비용(워커/Redis/DB)과 API 한계 내 동시성 계획에 따른 비용 모델이 필요.

## 핵심 리스크 TOP 3
1. **Instagram API 제약**: Business Discovery의 like_count 미제공 가능성으로 참여율 지표 신뢰 저하. 댓글 기반 대체는 편향 가능.
2. **GTM 공백**: 초기 고객 획득 채널·퍼널 미정으로 CAC 상승 및 0→1 지연.
3. **가격·가치 불일치**: SMB 예산대 대비 요금·가치 증명 부재로 유료 전환 저조 위험.

## 지금 당장 검증해야 할 1가지
> 유료 의향/가격 검증: 타깃 8–10개 브랜드와 문제–해결 데모 후 월 구독 가격대(Starter/Pro), 필수 기능 세트, 기대 성과(시간 절감/미스매칭 감소)의 지불 의향을 수치로 확보(예: LOI 또는 사전예약 3건 이상).

## 보완 제안 요약
- **가격/WTP**: 경쟁사 벤치마크(Starter ₩15–30만, Pro ₩45–90만) + 10개 인터뷰로 포지셔닝 확정.
- **GTM**: 패션 커뮤니티·제조/도매 파트너 공동 세미나, 레퍼런스 고객 3팀 파일럿(무상 2주→유료 전환).
- **지표 설계**: 베타에서 분석시간, 유료 전환율, 잔존율(3/6개월), CAC Payback 가정 수립.
- **제품 리스크 완화**: like_count 대체 로직 투명화(배지/한계 고지), 점수 산정 근거 노출로 신뢰 확보.
